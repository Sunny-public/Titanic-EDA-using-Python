# Titanic-EDA-using-Python
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns, trends, and key factors influencing passenger survival.

Using Python, Pandas, Matplotlib, and Seaborn, the dataset is analyzed through statistical summaries and visualizations such as histograms, boxplots, scatterplots, and correlation heatmaps.

🔍 Key Highlights:
Data understanding using .info(), .describe(), and .value_counts()
Handling missing values and identifying data inconsistencies
Visual analysis using multiple plots (histograms, boxplots, pairplots)
Correlation analysis to identify relationships between features
Insights on survival trends based on class, fare, age, and gender
📊 Key Insights:
Passengers in higher classes had better survival rates
Fare shows a strong relationship with survival probability
Missing values in Age and Cabin require preprocessing
Data reveals clear patterns useful for predictive modeling
